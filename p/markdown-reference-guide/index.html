Math

You start and end math using dollar signs. If you use on, you get inline math, and if you use two, you get “display” math, which means you get the math on a single line and centered.

This is inline math: $\int_a^b x,\mathrm{d}x$.

This is display math: $$\sum_{j=1}^N \frac{1}{j^2}$$

Pandoc supports a large subset of math you can write in LaTeX. For example, here is more complex equation. $$ d(i,j) = \begin{cases} ,i & j = 0 \ ,j & i = 0 \ ,\min\begin{cases} d(i-1,j-1) + \mathrm{1}(i\neq j) \ d(i,j-1) + 1 \ d(i-1,j) + 1 \end{cases} & i>0,j>0 \end{cases} $$
